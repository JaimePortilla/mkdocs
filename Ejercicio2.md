# Ejercicio 2: Jekyll con el tema Lagrange
# 1. Incorporar el tema Lagrange a mi cuenta de GitHub

Para trabajar con el tema Lagrange, primero debemos acceder al repositorio original en GitHub creado por LeNPaul. Desde allí, realizamos un fork para copiar el proyecto a nuestra propia cuenta de GitHub.
Una vez tengamos el fork en nuestro perfil, lo descargamos en la máquina virtual usando git clone junto con el token correspondiente, de forma que el repositorio quede enlazado como remoto.
